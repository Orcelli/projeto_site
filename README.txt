# projeto_site
Projeto de site primeiro semestre TSI

Linguagem de Marcação e Estilização

Objetivo:
Ter cinco (5) páginas;
Ter um texto de abertura do site e os links para acessar as suas demais páginas;
Formatar as páginas de tal forma que chame a atenção dos usuários;
Criar um formulário para os usuários enviarem suas opiniões sobre o site;
Criar uma tabela para demonstrar alguma informação importante, formate essa tabela;
Criar uma lista ordenada  para demonstrar alguma informação importante;
Criar um link absoluto para acessar outro site relacionado com o assunto.

prazo: 16/09/2018
